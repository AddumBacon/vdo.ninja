# \&chatbutton

Sender-Side Option! ([`&push`](../../source-settings/push.md))

## Aliases

* `&cb`
* `&chat`

## Options

Example: `&chatbutton=0`

| Value | Description           |
| ----- | --------------------- |
| `0`   | hides the chat button |
| `1`   | shows the chat button |

## Details

Adding `&chatbutton=0` or `1` hides or shows the chat button on the control bar.
