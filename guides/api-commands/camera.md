# camera

Sender Option! ([`&push`](../../source-settings/push.md))

| Value    | Description    |
| -------- | -------------- |
| true     | Turns mic on   |
| false    | Turns mic off  |
| "toggle" | Toggles on/off |

### CODE EXAMPLE

```
iframe.contentWindow.postMessage({ "camera": true });
```
