# sendChat

General Option! ([`&push`](../../source-settings/push.md), [`&view`](../../advanced-settings/view-parameters/view.md), [`&scene`](../../advanced-settings/view-parameters/scene.md))

| Value    | Description  |
| -------- | ------------ |
| (string) | Chat message |

### CODE EXAMPLE

```
iframe.contentWindow.postMessage({ "sendChat": "Hello" });
```
