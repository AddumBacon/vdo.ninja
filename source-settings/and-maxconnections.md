# \&maxconnections

Sender-Side Option! ([`&push`](push.md))

## Aliases

* `&mc`

## Options

Example: `&maxconnections=5`

| Value               | Description                     |
| ------------------- | ------------------------------- |
| (any integer value) | Will set maximum to given value |

## Details

Limits total of view and push connections to the given value.
