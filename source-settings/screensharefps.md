# \&screensharefps

Sender-Side Option! ([`&push`](push.md))

## Aliases

* `&ssfps`

## Options

Example: `&screensharefps=60`

| Value           | Description                        |
| --------------- | ---------------------------------- |
| (integer value) | will target this framerate. Eg: 60 |

## Details

Set a target FPS for your screen share.&#x20;
